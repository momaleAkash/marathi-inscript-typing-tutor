# GCC TBC Marathi Inscript Typing Tutor

A desktop typing tutor application built specifically for the **GCC TBC (Government Certificate in Computer Typing - Basic Course)** Marathi Inscript examination conducted by Maharashtra Civil Services.

---

## Motivation

I couldn't find any proper tool online for practicing Marathi Inscript typing as per GCC TBC standards. Offline coaching classes were expensive and inaccessible. Existing software either didn't follow the actual examination guidelines or lacked the specific requirements needed for Maharashtra civil services.

So I decided to build this tool myself — **accurate, free, and aligned with the official GCC TBC Inscript examination pattern**.

---

## Features

- ✅ Follows **GCC TBC Inscript** examination guidelines strictly
- ✅ Practice lessons matching actual exam content
- ✅ Real-time typing speed and accuracy tracking
- ✅ Keyboard layout visualization for Marathi Inscript
- ✅ Works **offline** — no internet required
- ✅ Cross-platform: **Linux** & **Windows**

## How to Use

1. Download the appropriate file for your OS
2. Run the application
3. Select a lesson and start practicing
4. Track your speed (WPM) and accuracy in real-time
